# Ruby 开始

1、中文编码

```ruby
#!/usr/bin/ruby -w
# -*- coding: UTF-8 -*-

puts "你好，世界！";
```

**注**：Ruby2.0~2.6 版本中文注释有可能需要使用: `# -*- coding: GBK -*-`

2、注释

```ruby
#!/usr/bin/ruby -w
# -*- coding: UTF-8 -*-

#这是一个单行注释
#a = 3

#这是一个多行注释
=begin
a = [0,1,2,3]
hashPara = {:name=>"name1",:number=>"num1"}
=end
```
