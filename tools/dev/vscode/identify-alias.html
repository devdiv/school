# VSCode 识别 alias

很多时候为了避免复杂的层级关系，常常使用 webpack 的 alias 功能来简化路径，但这也产生了新的问题，无法实时解析文件中的内容，导致缺乏提示。

在 VSCode 官网也提供了该项的说明在新窗口打开。

# 配置文件

在项目根目录下新建 jsconfig.json：

{
  "compilerOptions": {
    "target": "es6",
    "baseUrl": ".",
    "paths": {
      "@/*": ["src/*"]
    }
  },
  "exclude": ["node_modules", "dist"],
  "include": [
    "./src/**/*.vue",
    "./src/**/*.js"
  ]
}
在线编辑文档 在新窗口打开
